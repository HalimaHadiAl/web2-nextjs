## Cara Install Project

1. Clone repository

git clone https://github.com/HalimaHadiAl/web2-nextjs.git

2. Masuk ke folder project

cd web2-nextjs/my-app

3. Install dependency

npm install

---

## Menjalankan Project

Jalankan server development:

npm run dev

Kemudian buka browser:

http://localhost:3000

---

## Struktur Folder

app/

- profile/page.js → halaman Profile
